# Concept

## Core Definition

- execution = 戦略を現実へ変換すること
- 計画は目的ではなく前進の道具
- 実行管理 = 改善ループ
- KPT = システム改善
- YWT = 学習の言語化
- PDCA = 継続改善
- Companion Layer = 並走支援
- execution-harness = 実行・測定・改善・再始動OS

## Why This Exists

戦略だけでは成果は生まれません。execution-harness は、戦略を日々の行動に翻訳し、測定と学習で再設計を続けるための実行基盤です。

## Design Principle

1. 完璧な計画より、再開可能な運用
2. 叱責より、現状整理と次の一歩
3. 記録より、改善に使える記録
4. 進捗管理より、前進継続の設計

## Next Step Rule

すべての計画・レビュー・対話は、最後に以下を明確化します。

- いつ
- 何を
- どの粒度で
- どこまでやれば完了か

この4点が定義されていない出力は未完了とみなします。
